# RSA_Key_Generator

> Generate random RSA Keypair with user seed

author: Nguyen Thanh Dat 1612088

### Install dependencies

```
pip install -r requirements.txt
```

### Test program

```
python -m pytest
```

### Run program

```
python src/rsa_key_gen.py [seed]
```

### Remove cache

```
py3clean .
```
